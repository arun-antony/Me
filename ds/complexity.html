Complexity

Time Complexity:

  1. Measure of time required to execute a code.
  2. Represented using Big O notation : O(ndeg). Order of deg
  3. If the code is represented as a function of f(n), then the complexity is the degree of the polynomial f(n)

Examples

Case 1: Complexity for code with a simple function with no loop is O(1) or const

Code which does some operation that has no repetition. A simple addition function.

      
        
      int sum(int a, int b){

          int sum = a+b; //Executes 1 time
          
          return sum; //Executes 1 time
          
      }

      //The number of executions for above function can be represented as f(n) = 2
      //So the complexity is degree of f(n) which is 0
      //Hence the complexity is O(n0) which is same as O(1)

Case 2: Complexity for code with a single depth loop is O(n)

A function which iterates through a single dimension array and does some operation.
Complexity is for worst case scenario is when it goes all the way to the end of the array.
If there are multiple loops in a function that are NOT NESTED, the complexity will be still the same.

      
        
      int sum(int array[]){

          int sum = 0; //Executes 1 time
          int n = array.size(); //Execute 1 time
          for(int i = 0; i < n; i++){
              sum += array[i]; //Executes 'n' times
          }
          return sum; //Executes 1 time

      }

      //The number of executions for above function can be represented as f(n) = n+3
      //So the complexity is degree of f(n) which is 1
      //Hence the complexity is O(n1) which is same as O(n)

Case 3: Complexity for code with a nested depth loop is O(nnum of nesting)

A function which iterates through a matrix and does some operation.
Or iterates through an array and for each item, iterates through every item in that array 

      
        
      int sum(int array[]){

          int sum = 0; //Executes 1 time
          int n = array.size(); //Execute 1 time
          for(int i = 0; i < n; i++){ //Executes 'n' times
             
            for(int j = 0; j<n; j++){ //Executes 'n*n' times
               //Does some operation
            }
          }
          return sum; //Executes 1 time

      }

      //The number of executions for above function can be represented as f(n) = n2+n+3
      //So the complexity is degree of f(n) which is 2
      //Hence the complexity is O(n2)
